Learn HTML: Elements

HTML Tags

Tag Name Description
<li> List Item Element The list item element create list items inside ordered lists and unordered lists
<video> Video Element The video element embeds a media player for video playback.
<em> Emphasis Element The emphasis element emphasizes text and browsers will usually italicize the emphasized text by default.
<div> Div Element The Div element is used as a container that divides an HTML document into sections and is short for “division”
<br> Line Break Element The line break element will create a line break in text.
